# ComfyUICarDriverImage

## Files

```
ComfyUICarDriverImage.json - ComfyUI workflow
ComfyUICarDriverImage.png - sample output image with car and driver
control image.png - car outline image
driver outline.png - driver outline image
driver_seat_mask.png - driver seat mask
```
---

## How to run

1. Open ComfyUI Cloud and load ComfyUICarDriverImage.json
2. Upload assets in corresponding "Load Image" nodes and click Run
3. View results in the Generated Files panel
